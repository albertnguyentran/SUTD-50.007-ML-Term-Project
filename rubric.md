# Grading Rubric

## Task 1: Logistic Regression from Scratch (10 marks)

| Marks | Criteria |
|-------|----------|
| 0 | No submission, or used sklearn logistic regression / any pre-defined logistic regression or multi-class wrapper |
| 1–4 | Binary Logistic Regression correctly implemented, but no multi-class variant |
| 5–7 | Multi-class Logistic Regression implemented with minor errors; model compiles but has issues training on the dataset |
| 8–10 | Perfect implementation of multi-class Logistic Regression; successfully trained on train set with performance comparable to sklearn's Logistic Regression |
